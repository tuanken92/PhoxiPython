# Step 1: Run conda
```bash
  conda activate phoxi
```

# Step 2: Run program
```bash
  python .\main.py
```

- Note: recheck log and use command
- q: quit program
- r: reconnect camera
- t: trigger camera
- e: close camera
- D: debug on
- d: debug off
- S: saved detected inference
- s: don't save detected inference
- load: load model Deeplearning
- ./image_folder/image_name.jpg|bmp : inference 


# Step 3: Exit program
- Press q character then enter


# Note
- Photoneo takes 5-10s to start when camera reconnect
